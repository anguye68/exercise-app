Getting Started

Welcome to the Exercise App! This page will give you a basic overview of how to use the application. To get started, all you have to do is add an exercise! You can do so by pressing the large green button that says "Add Exercise" on the main page to add your first exercise. You will need to fill out some basic information, including the date that you performed the exercise, the name of the exercise, the weight, the number of repetitions performed, and the number of sets performed. The add exercise window will have tooltips available to explain further what each of these mean, including examples! Just hover over the icon for an explanation.
